### 👨‍💻 À propos de moi

Développeur Frontend & Backend avec plus de 4 ans d'expérience, j'aborde chaque projet comme un partenariat. Ma mission est de transformer une vision en une réalité digitale performante. J'aime construire des architectures solides côté backend tout en sculptant des interfaces réactives et esthétiques côté frontend. La qualité du code, la maintenabilité et la performance sont au cœur de ma démarche.

-   🔭 Je travaille actuellement sur des applications web internes critiques et des projets personnels innovants.
-   🌱 J'explore constamment les nouvelles possibilités offertes par l'IA et les architectures microservices.
-   💬 Discutons de **Vue/Nuxt, React/Next, Node.js, Laravel** ou de tout sujet lié à la tech !

### 🌟 Projet Mis en Avant : Mon Portfolio Interactif

J'ai conçu mon propre portfolio comme une démonstration de mon savoir-faire. Il ne s'agit pas d'un simple site statique, mais d'une application Nuxt.js complète. Il est la preuve vivante de ma philosophie de développement.

-   **Backend Intégré :** Une API sécurisée gère le formulaire de contact.
-   **Animations & UX :** L'interface est enrichie d'un fond de particules interactif et de micro-animations pour une expérience mémorable.
-   **CI/CD :** Le déploiement est entièrement automatisé avec GitHub Actions et Vercel.

**[➡️ Explorez le code source de mon portfolio](https://github.com/awarris/portfolio-background-animate)**
